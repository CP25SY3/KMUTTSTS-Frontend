# shared

This folder contains shared React components that are reused across multiple features or pages, such as layout, navigation, or common UI elements.

## Available Components

- `UniversalVideoCard.tsx` - Universal video/content card component
- `Sidebar.tsx` - Sidebar navigation component  
- `SearchBar.tsx` - Search bar component

## Example

```
/shared/
  UniversalVideoCard.tsx  // Universal video card for all content types
  Sidebar.tsx            // Sidebar navigation component
  SearchBar.tsx          // Search bar component
```
